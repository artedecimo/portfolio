Web Design Agreement

  • Client/Web Designer Relationship

    • Web Designer is working as an independent contractor for Client, not an employee.
    • Web Designer is only specialized in Front-End Web Design. Client shall seek other resources regarding Back-End programming or scripting.
    • This Agreement between Client and Web Designer shall only become valid once submitted by Client AND approved by Web Designer.

  • Web Hosting

    • All web hosting and domain accounts have a separate third-party subscription with a domain/hosting provider.
    • Client may choose to setup their own domain/hosting service, or Web Designer may setup domain and/or hosting for Client, which would increase the cost of the project, if not already included in the Scope of Work.

  • Project Duration

    • The project duration is determined by the Client contributing to the workflow and delivering content to Web Designer in a timely manner. Any delays in approvals or responses from the Client will significantly affect the projected duration, with the possibility of the selected End Date being postponed.
    • By agreeing to these terms, Client acknowledges that Web Designer shall determine how many revisions will be made without affecting the project duration. On the other hand, if the Web Designer is unable to meet project deadlines, the project duration will be adjusted accordingly.

  • Maintenance

    • If you selected Ongoing Maintenance as your Project Goal, setting an hourly rate is recommended. Any payment(s) for Ongoing Maintenance, will be due in installments at the proposed rate in the Contract.

  • Third-party services or subscriptions

    • Any third-party subscriptions (ex: stock photography) and/or additional materials and services used for the project, shall be provided by Client.

  • Custom Assets

    • Custom Assets are defined as anything created by Web Designer not provided by Client including but not limited to (animated GIFS, graphics, animations, web templates, printed materials etc).
    • Custom Assets are considered ("original content") created by Web Designer.

  • Ownership

    • Web Designer retains the right to use, display or create derivative works from all ("original content") created by Web Designer for Client.
      • This is mainly, but not exclusively, for the purpose of displaying portfolio work on this site.

    • Client's website should be compatible with the most recent browser versions* of:

      • Chrome
      • Internet Explorer .

    • Web Designer is not responsible for any costs, fees, suits, claims, penalties or otherwise, related to Client's use of their website.

    • Work begins on the selected Start Date or once Web Designer has reviewed and approved the Contract, whichever is later. Any Payment Options other than One-Time Payment will be invoiced. If Client selects the Payment per Task or the One-Time payment option, Client agrees that payment will be due BEFORE work begins. Online Payments will only be processed thru Paypal .
    • All invoices are due as soon as they are recieved. If Web Designer doesn't recieve payment within 7 days of the invoice date, it will be assumed, at Web Designer's own discretion, that Client terminated the project.

    • If Client terminates the project, all prior work done by Web Designer up until the termination date will be billed to Client. Once project is terminated, Web Designer will no longer be responsible for providing additional services to Client.
    • If the project is terminated because of Web Designer's inability to complete their tasks, and Client has paid in advance for services, the amount of a refund given to Client shall be determined based on hours spent and/or how close the project was to completion.
